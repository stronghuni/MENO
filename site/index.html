회의록은
알아서 챙길게요.

느긋한 친구처럼, MENO가 회의를 녹음하고 한국어로 받아 적고 깔끔한 회의록까지 써둡니다.

마이크 녹음도, 이미 가진 mp3·wav·mp4 파일 업로드도. 회의록은 토큰 단위로 실시간으로 채워지고, 음성·전사·요약 전 과정이 당신의 맥 안에서만 돌아갑니다.

기능

회의 하나에 필요한 전부

전사부터 회의록, 관계 그래프, 일정 관리까지 — 클라우드 없이 맥 한 대로.

로컬 STT

Whisper Large-v3 Turbo로 한국어를 정밀하게 받아 적습니다. Metal GPU 가속, 1시간+ 회의는 자동 청크 처리.

기존 파일 업로드

mp3·wav·m4a·flac·mp4·mov·avi·mkv까지 — 이미 녹음해 둔 회의 파일을 드래그하면 자동으로 16kHz로 변환·전사·회의록 작성.

실시간 회의록 작성

Qwen 2.5-7B가 TL;DR·안건·논의·결정·액션 아이템을 표준 포맷으로 토큰 단위 스트리밍 — 회의록이 채워지는 모습을 그대로 봅니다. 무음 환각 방지 가드 포함.

관계 그래프

회의가 쌓이면 참석자·주제가 겹치는 회의끼리 자동으로 이어집니다. 옵시디언 스타일 그래프로 한눈에.

캘린더 · 일정

회의록에서 다음 회의를 알아서 찾아 캘린더에 등록하고, 시작 10분 전 알림을 띄웁니다.

Notion · Jira 연동

완성된 회의록을 Notion에 한 행으로 업로드하고, 액션 아이템을 Jira 이슈로 내보냅니다. 토큰은 Keychain에.

사용법

받고, 모델 한 번 내려받고, 녹음

설정은 처음 3단계뿐. 그다음부터는 버튼 하나면 회의록이 나옵니다.

1

다운로드 & 설치

DMG를 열고 MENO를 Applications로 드래그. 처음 한 번 격리 해제 명령만 실행하면 됩니다.

2

모델 자동 다운로드

첫 실행 시 온보딩이 Whisper + Qwen 모델(약 6.3GB)을 내려받습니다. 끊겨도 이어받기.

3

녹음 또는 파일 업로드

마이크로 녹음하거나, 가진 mp3·mp4 파일을 드래그. 끝나면 전사 → 회의록 → (켜둔 경우) Notion 업로드까지 자동.

$ xattr -dr com.apple.quarantine /Applications/MENO.app

미공증 빌드라 처음 열 때 한 번만 격리(quarantine) 속성을 풀어주면 이후로는 그냥 실행됩니다.

설치 가이드

처음 한 번만, 차근차근

받아서 바로 쓰기

  1. DMG 다운로드 — 위에서 맥 종류에 맞는 파일을 받습니다.
  2. 설치 — DMG를 더블클릭해 열고 MENO를 Applications 폴더로 드래그.
  3. Gatekeeper 한 번 풀기 — 미공증 빌드라 처음 열 때 막힙니다. 터미널에 아래 한 줄을 실행하세요.
$ xattr -dr com.apple.quarantine /Applications/MENO.app
  1. 마이크 권한 — 첫 녹음 시 접근을 허용합니다.
  2. 모델 다운로드 — 온보딩이 Whisper(1.6GB) + Qwen(4.7GB)을 자동으로 받습니다. 끝나면 바로 녹음.
이 명령은 macOS가 다운로드 파일에 붙이는 "격리" 표시만 지웁니다. 앱을 수정하지 않으며, 정식 공증 빌드를 받으면 필요 없는 단계입니다.

소스에서 빌드 개발자용 ✎

$ git clone https://github.com/stronghuni/MENO.git && cd MENO
$ npm install
$ npm run dev

Node 20.x / 22.x LTS 권장 · Xcode CLT 필요(xcode-select --install). DMG 빌드는 npm run build:mac.
